HTML

Ideas

- el inicio de una etiqueta es normal, en otras palabras lleva los dos caracteres y dentro de estos se encuentra el nombre. - para el cierre de la etiqueta antes de escribir el nombre de la etiqueta debemos escribir una diagonal (/) - entre el inicio y el fin de la etiqueta debe estar el contenido

Apuntes

- HTML (HyperTex Markup Language) componente mas basico de la web, defime el sigmificado y la estructura del contenido web. - La gran mayoria de las etiquetas deben tener un inicio y un fin. - Se necesitan ciertos simbolos para etiquetar todo

Resumen

en resumen el HTML es de lo que se compone la web, al menos de forma basica, es una forma de realizarlas y tambien tienes que ser muy cuidadoso con los detalles ya que una letra mal (faltante, minuscula,mayuscula, mal escrita) puede icasionar un problema grande en toda tu pagina cono el nulo funcionamiento de esta

css

Ideas

hay 3 formas de agregar CSS al HTML: - hoja de estilo incrustada - hoja de estilo en linea - hoja de estilo externa cada una funcionando de manera distinta con sus respectivas ventajas y desventajas.

Apuntes

- mecanismo simple para dar estilo o diseno a a los sitios web. - se les da la apariencia que uno desee utilizandolo, por ejemplo: fuente, colores, espacio, margen, etc..

Resumen

La conclucion de esto es el CSS es un metodo para dar diseno a la pagina o lo que seria "la cobertura linda del pastel"

Texto en Html

Ideas

la pagina se divide en ciertas cosas para que todo encaje y pueda funcionar, los titulos se colocan en un h1 los subtitulos en un h2 y los textos (y sus resaltados) en una p.

Apuntes

Se usa un h1 por pagina - los parrafos se encuentran en el elemento p - las listas se dividen en dos: ordenadas y desordenadas - lo que se necesita para resaltar parrafos o palabras.

Resumen

el texto en HTML no me es diferente al texto en word solo hay que anadir etiquetas

Enlaces en Html

Ideas

Su sintaxis es la sig: apertura + atributo a definir + destino + parte visible para el usuario + cierre.

Apuntes

Los links pueden ser cualquier cosa desde un texto, una imagen, un video, un boton, etc. por defecto los links aparrecen de tres maneras en todos los navegadores: - enlace no visitado - enlace visitado - enlace activo.

Resumen

los links pueden insertarse en el HTML y te guian a lo que el servidor decida, pero tambien se pueden ligarse al CSS asi se juntan ambos.

Imagenes en html

Ideas

su sintaxis comienzan con img scr = y buscas la carpeta en donde se encuentra ( es importante teberlas en un folder cercano al index y al CSS si no no aparecen).

Apuntes

las imagenes pueden estar en el mismo folder, en diferentes y online.

Resumen

las imagenes en la web no solo sirven por cuestiones esteticas, ayudan a lus buscadores a indexar las paginas, brindan a los usuarios confianza y una buena experiencia de usuario.
